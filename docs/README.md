# scilus/sf-tractomics: Documentation

The scilus/sf-tractomics documentation is split into the following pages:

- [Usage](https://scilus.github.io/sf-tractomics/user_guides/usage/)
  - An overview of how the pipeline works, how to run it and a description of all of the different command-line flags.
- [Input](https://scilus.github.io/sf-tractomics/user_guides/inputs/)
  - An overview of what kind of input you should prepare to run sf-tractomics.
- [Output](https://scilus.github.io/sf-tractomics/user_guides/outputs/)
  - An overview of the different results produced by the pipeline and how to interpret them.
